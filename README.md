- [ ] HOAS 求值.

- [ ] 代数数据类型和模式匹配
- [ ] 模式完全性检查
- [ ] 无K的模式匹配
- [ ] REPL
- [ ] 模块系统
- [ ] 运算符
- [ ] 停机检查
- [ ] 归纳类型的极性检查
